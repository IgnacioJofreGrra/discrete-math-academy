# Politica de Seguridad

## Versiones Soportadas

Este proyecto actualmente soporta la ultima version disponible en la rama `main`.

## Reportar una Vulnerabilidad

Si descubres un problema de seguridad, no publiques detalles de explotacion en issues publicos.

Por favor reportalo por uno de estos canales:

- GitHub Security Advisories (recomendado)
- Contacto privado al mantenedor: Ignacio Jofre Guerra

Incluye:
- Archivo o funcionalidad afectada
- Pasos para reproducir
- Impacto potencial
- Mitigacion sugerida (si aplica)

El mantenedor confirmara la recepcion del reporte lo antes posible y compartira actualizaciones de remediacion.
