# Política de Seguridad

## Versiones Soportadas

Solo la rama `main` recibe parches de seguridad activos.

| Versión | Soporte de seguridad |
| --- | --- |
| `main` | Sí |
| Cualquier otra rama o snapshot histórico | No |

## Reportar una Vulnerabilidad

Si identificas una posible vulnerabilidad de seguridad:

- No abras un issue público.
- No publiques detalles de explotación, PoC ni trazas sensibles en canales públicos.

Reporta de forma privada a:

`issues@alvy.site`

Incluye, en lo posible:

- Componente o archivo afectado.
- Pasos para reproducir.
- Impacto potencial.
- Mitigación sugerida (opcional).

## SLA de Respuesta

- Acuse de recibo inicial en un máximo de 72 horas.
- Si el reporte es válido, el equipo de mantenimiento compartirá estado de triage y progreso de remediación.

## Divulgación Coordinada

Este proyecto aplica divulgación coordinada:

1. Recepción y validación privada del reporte.
2. Evaluación de riesgo e impacto.
3. Desarrollo y verificación del fix.
4. Divulgación pública coordinada cuando exista una mitigación razonable.
